If you are building an Angular enterprise web application, soon you'll notice navigating through various views are required. Angular routing provides capabilities for displaying different views and this technique simplifies the navigation for large web applications. The router can handle optional parameters for views to help figuring out as which content to display. Router keeps a log in browser's history for ease of going forward and back between views.
